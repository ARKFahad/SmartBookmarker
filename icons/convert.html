How to Convert SVG to PNG Icons

  1. Open this HTML file in a web browser
  2. Right-click on each icon size below
  3. Select "Save image as..."
  4. Save with the appropriate filename (icon16.png, icon48.png, icon128.png)
  5. Place the PNG files in the icons folder
